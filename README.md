🏛️ Student Parliament - Ananda Balika Vidyalaya
This is a formal web project I developed for my school, Ananda Balika Vidyalaya, during my Grade 12 studies. The website serves as a digital platform for the Student Parliament to share updates, announcements, and information with the student body.

Live Website: [View the Student Parliament Site](https://taniyasewwandii.github.io/StudentParliamentOfABV/)

📖 Project Purpose
The goal of this project was to digitize the school's parliamentary activities. It allows students and teachers to:

View the latest school news and parliament updates.

Learn about the roles and responsibilities of the Student Parliament.

Access information about school events and leadership.

🛠️ Technical Stack
To ensure the website was lightweight and easy to host, I used:

HTML5: For structured content and semantic layouts.

CSS3: For custom styling, including the school's branding and colors.

Responsive Design: Optimized for viewing on both school computers and mobile devices.

✨ Key Features
Navigation Menu: Easy access to different sections of the parliament.

Announcement Section: A dedicated space for important school notices.

Visual Gallery: Highlighting school events and parliamentary sessions.

School Branding: Designed to align with the official identity of Ananda Balika Vidyalaya.
